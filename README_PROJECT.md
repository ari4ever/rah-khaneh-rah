# Civil Engineering Website

This project is a Next.js application for a Civil Engineering consultancy and construction firm.

## Features Implemented
- **Design System**: Professional Black, White, and Beige theme.
- **Home Page**: Hero section, Featured Services, Featured Products.
- **Services Page**: Detailed list of consulting, design, and execution services.
- **Shop Page**: Product grid for construction materials and tools.
- **Cart Page**: Basic cart layout.
- **Login Page**: Login form.
- **Responsive Design**: Mobile-friendly navigation and layout.

## How to Run
1. Install dependencies (if not already successful):
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Note on Disk Space
The system encountered `ENOSPC` (No space left on device) errors on the C: drive during the setup. This might affect `npm install` or `npm run build`. Please ensure you have enough space on your C: drive or configure npm to use a different cache location if issues persist.
